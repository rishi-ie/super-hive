# Mumbrane Desktop

A local-only desktop code editor for AI agents. No auth, no cloud, no accounts — just run it.

## Tech Stack

- Electron + React + TypeScript
- Bun + Turborepo
- TailwindCSS + shadcn/ui
- Drizzle ORM + SQLite (local)

## Dev

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
```
